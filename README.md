## Install using homebrew
* pyenv
* direnv
* httpie

## Configure direnv to load in your shell
https://direnv.net

## Install Python
`pyenv install 3.9.1`

## Create .envrc file
`cp envrc.sample .envrc`

## Look at examples in the httpbin folder
`curly httpbin/get.status.yml`

`curly httpbin/post.something.yml`
